A Linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup
Language, que significa Linguagem de Marcaçao de Hipertexto)
é uma linguaguem de marcação utilizada na construção de páginas
Web. Documentos HTML podem ser interpretados por navegadores.


Alguns recursos da linguagem HTML vistos até este momento.

Recursos Tags Associados Descrição
Link <a> Criar link entre páginas
Lista <ol> Criar uma lista ordenada
<ul> Criar uma lista não-ordenada
Imagens <img> Inserir Imagem
